3.Explain Local Scope, Block Scope, Functional Scope and Scope Chain in javascript

Local scope:Local Scope occurs when you create a variable inside a function. By doing that, the visibility and accessibility of the variable is only allowed within that function.

Local scope: Scope inside which we define variables as well as functions and restirct the all the types of variables methods to that scope only

Block scope: A block scoped variable means that the variable defined within a block will not be accessible from outside the block. A block can reside inside a function, and a block scoped variable will not be available outside the block even if the block is inside a function

This will restrict the let and const variables to the block only

Function Scope: When a variable is declared inside a function, it is only accessible within that function and cannot be used outside that function

Scope Chain: means that one variable has a scope (it may be global or local/function or block scope) is used by another variable or function having another scope (may be global or local/function or block scope)
